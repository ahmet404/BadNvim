- Statusline with [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim)
- Bufferline with [bufferline.nvim](https://github.com/akinsho/bufferline.nvim)
- Breadcrumbs with [dropbar.nvim](https://github.com/Bekaboo/dropbar.nvim)
- Manage multiple terminal with [toggleterm.nvim](https://github.com/akinsho/toggleterm.nvim)
- Dashboard with [alpha-nvim](https://github.com/goolord/alpha-nvim)
- Git manage with [gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim) & [neogit](https://github.com/NeogitOrg/neogit)

- Autocomplete with [nvim-cmp](https://github.com/hrsh7th/nvim-cmp)
- Snippets code with [LuaSnip](https://github.com/L3MON4D3/LuaSnip)
- LSP with [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)
- Formatting code with [none-ls.nvim](https://github.com/nvimtools/none-ls.nvim)
- Fuzzy finder with [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim)
